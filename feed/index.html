<pre><code>#用户分3种 1、root用户，即超级管理员 2、拥有root 权限的用户，可以执行sudo命令 3、普通用户 #用户文件与配置 /etc/passwd /etc/shadow</code></pre> <p><strong>1、users命令</strong></p> <pre><code>users #查看当前在线的用户名称列表</code></pre> <p><strong>2、adduser命令</strong></p> <pre><code>adduser demo #添加账号</code></pre> <p><strong>3、useradd命令</strong></p> <pre><code>useradd demo #添加用户，默认家目录在/home/目录下demo目录，与用户名一致。 passwd demo #设置密码，默认添加用户之后没设置密码不允许登录。 useradd -u 2333 demo #添加用户并指定用户id ，一般情况很少使用。 useradd -d /home/testuser demo #添加用户并指定家目录，一般情况很少使用。 useradd -G xxx,xxx demo #添加用户并加入多个附加组。</code></pre> <p><strong>4、usermod命令</strong></p> <pre><code>usermod -L demo #锁定用户，不允许登录，相当于手动修改/etc/shadow文件中用户密码前加入!，一个感叹号。 usermod -U demo #解除锁定</code></pre> <p><strong>5、userdel命令</strong></p> <pre><code>userdel demo #删除用户，不删除用户家目录数据。 userdel -r demo #删除用户并删除用户家目录。 userdel -rf demo #强制删除用户无论是否登录，并删除用户家目录。</code></pre> <p><strong>6、passwd命令</strong></p> <pre><code>passwd -l demo #锁定用户，不允许登录，相当于手动修改/etc/shadow文件中用户密码前加入!!，双感叹号。 passwd -u demo #解除锁定</code></pre> <h2>用户组</h2> <pre><code>#用户组分2种 1、初始组(默认组) 2、其他组(附加组) #用户组文件与配置 /etc/group /etc/gshadow</code></pre> <p><strong>1、groups命令</strong></p> <pre><code>groups #查看当前用户的用户组名称</code></pre> <p><strong>2、addgroup命令</strong></p> <pre><code>groupadd test #添加用户组</code></pre> <p><strong>3、groupmod命令</strong></p> <pre><code>groupmod -n new_test test #将test组名修改为new_test</code></pre> <p><strong>4、gpasswd命令</strong></p> <pre><code>gpasswd -a nick new_test #将nick用户添加到new_test组 gpasswd -d nick new_test #将nick用户从new_test组删除</code></pre> <p><strong>5、groupdel命令</strong></p> <pre><code>groupdel new_test #删除用户组</code></pre> <p>提示：每添加一个用户会自动创建一个用户组，称为默认组或初始组，这个组是不允许删除的，默认组一般不修改最好，用户<code>id</code>也一样。</p> <h2>相关命令</h2> <p><strong>1、sudo命令</strong></p> <pre><code>sudo vim /etc/passwd #以另一个用户身份执行命令，通常是root用户</code></pre> <p>提示：普通用户添加执行<code>sudo</code>的方法，在<code>/etc/sudoers</code>文件中添加<code>username ALL=(ALL) ALL</code>即可。</p> <p><strong>2、su命令</strong></p> <pre><code>su #切换到root用户，仅切换路径 su - #切换到root用户，环境变量和路径一起切换 su - nick #切换到nick用户，环境变量和路径一起切换</code></pre> <p><strong>3、whoami命令</strong></p> <pre><code>whoami #查看当前用户的名称</code></pre> <p><strong>4、id命令</strong></p> <pre><code>id #查看当前用户的id 、组id 、附加组id id -un #查看当前用户的名称 id -g #当前用户的组id id -G #当前用户的附加组id id -u root #查看指定用户的id id root #查看指定用户的id gid groupsid</code></pre> <p><strong>5、who命令</strong></p> <pre><code>who #查看当前的在线用户、终端号、IP、登录时间 who -q
